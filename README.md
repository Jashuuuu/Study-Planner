# Project README

## Overview
This project appears to be a web application with user authentication and admin dashboard features. It uses Python for backend logic and HTML/CSS/JS for the frontend.

## Project Structure
- `app.py`: Main application entry point
- `config.py`: Configuration settings
- `db.py`: Database connection and setup
- `dp_logic.py`: Business logic
- `models.py`: Database models
- `seed_db.py`: Script to seed the database
- `requirements.txt`: Python dependencies
- `static/`: Static files (JS, CSS)
- `templates/`: HTML templates

## Setup Instructions
1. **Clone the repository**
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Set up the database:**
   - Configure your database in `config.py` if needed.
   - Run the seed script:
     ```bash
     python seed_db.py
     ```
4. **Run the application:**
   ```bash
   python app.py
   ```
5. **Access the app:**
   - Open your browser and go to `http://localhost:5000` (or the port specified in your app)

## Features
- User registration and login
- Admin dashboard
- Database seeding
- Static assets (CSS/JS)
